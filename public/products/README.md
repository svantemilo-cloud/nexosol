# Produktkort – bakgrundsbilder

Lägg dina bilder här så att de används som bakgrund på produktkorten:

- **solceller.png** – kortet "Solceller"
- **solcellsbatteri.png** – kortet "Solcellsbatteri"
- **laddbox.png** – kortet "Laddbox"

Rekommenderad storlek: minst ca 600×400 px (bilden skalas med `background-size: cover`).
Format: JPG eller PNG.
